# Learning Claude API

Part of the **[Building with the Claude API](https://anthropic.skilljar.com/claude-with-the-anthropic-api)** course

A collection of Jupyter notebooks for learning how to build with the [Anthropic Claude API](https://docs.anthropic.com/en/api).

## Notebooks

| # | Notebook | Description |
|---|----------|-------------|
| 000 | [Multi-Turn Exercise](000-multi-turn-exercise.ipynb) | Building multi-turn conversations with message history |
| 001 | [Chat Exercise](001-chat-exercise.ipynb) | Interactive chat loop with user input |
| 002 | [Math Tutor](002-math-tutor.ipynb) | Using system prompts to create a guided math tutor |

## Setup

```bash
pip install anthropic python-dotenv
```

Create a `.env` file with your API key:

```
ANTHROPIC_API_KEY=your-key-here
```
